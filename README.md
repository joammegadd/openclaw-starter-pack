# Pack v01 — Instalação (Docker Compose)

Este pack é o “starter” do OpenClaw rodando via **Docker Compose**, com:
- **Gateway** (websocket / API) exposto na porta **18789**
- **CLI** opcional (profile `cli`) para rodar comandos dentro do container
- Persistência via volumes para **config** e **workspace**

> ✅ Ideal para subir rápido e já ter uma base que dá pra evoluir nos próximos vídeos/packs.
